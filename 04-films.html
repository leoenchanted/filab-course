如何用代码"模拟"真实胶片的色彩科学

胶片为什么看起来不一样？

每种胶片都有独特的"性格"。Kodak Portra 擅长肤色，Fuji 偏绿，CineStill 有标志性的红色光晕。这些差异来自胶片的化学配方和显影工艺。

数字模拟的核心是：用数学公式复制这些化学特性。Film Lab 使用了四种主要技术：色彩矩阵、曲线映射、光晕效果和颗粒生成。

色彩矩阵变换

色彩矩阵 是最基础的色彩调整方式。通过 3x3 矩阵，我们可以重新混合红、绿、蓝三个通道的比例。

CODE 
// Kodak Portra 400 的色彩矩阵
matrix: [
  0.92, 0.05, 0.03,  // 新 R = 0.92*R + 0.05*G + 0.03*B
  0.00, 0.98, 0.02,  // 新 G = 0.00*R + 0.98*G + 0.02*B
  0.04, 0.08, 0.88   // 新 B = 0.04*R + 0.08*G + 0.88*B
]
PLAIN ENGLISH

注释：这是柯达 Portra 400 胶片的色彩矩阵配置

矩阵开始

第一行：新的红色通道 = 92%原红 + 5%原绿 + 3%原蓝

第二行：新的绿色通道 = 0%原红 + 98%原绿 + 2%原蓝

第三行：新的蓝色通道 = 4%原红 + 8%原绿 + 88%原蓝

矩阵结束

💡
Portra 的"肤色保护"

观察上面的矩阵：绿色通道几乎保持不变（98%），红色通道略微降低（92%），蓝色通道明显降低（88%）。这种组合让肤色更暖、更自然，是 Portra 胶片备受人像摄影师喜爱的原因。

曲线映射

曲线映射 让胶片可以对不同亮度区域做不同的处理。比如让暗部更实、亮部更柔，这就是胶片特有的"宽容度"。

CODE 
// Fuji Pro 400H 的曲线
curve: [
  [0,   0],   // 输入 0 → 输出 0（纯黑保持）
  [40, 38],   // 输入 40 → 输出 38（暗部微压）
  [128, 135], // 输入 128 → 输出 135（中间调提亮）
  [220, 232], // 输入 220 → 输出 232（亮部提升）
  [255, 252]  // 输入 255 → 输出 252（纯白微压）
]
PLAIN ENGLISH

注释：Fuji Pro 400H 胶片的曲线配置

曲线控制点数组开始

第1点：输入0(纯黑) → 输出0，保持纯黑

第2点：输入40(暗部) → 输出38，略微压缩

第3点：输入128(中间调) → 输出135，提亮

第4点：输入220(亮部) → 输出232，明显提升

第5点：输入255(纯白) → 输出252，略微压缩

数组结束

🎨
Fuji 的" airy look"

Fuji 胶片的特点是亮部特别通透。看曲线：220 的输入被映射到 232 的输出，这意味着亮部被整体提亮，产生那种"空气感"。同时纯白微压（252）防止过曝死白。

亮度阈值：只有超过85%亮度的高光才会产生光晕

强度：光晕效果的强弱程度（0.6=60%）

颜色偏移：RGB分别为1.8(红)、0.4(绿)、0.1(蓝)=橙色

小模糊半径：产生清晰的核心光晕

大模糊半径：产生弥散的边缘光晕

胶片颗粒的秘密

胶片颗粒不是简单的"噪声"，而是有结构的。亮部颗粒细、暗部颗粒粗，彩色胶片还有色度颗粒和亮度颗粒的区别。

CODE 
// 频谱颗粒生成（Spectral Grain）
buildSpectralGrain(width, height, intensity) {
  // 1. 生成随机噪声纹理
  const noiseTex = this.ensureNoiseTexture(...);

  // 2. 高通滤波提取细颗粒
  const fineTex = this.buildHighpassLuma(noiseTex, ...);

  // 3. 可选：模糊产生粗颗粒
  if (softness > 0.001) {
    const blurred = this.blurTexture(noiseTex, softness);
  }

  // 4. 按亮度混合（亮部颗粒少，暗部颗粒多）
  return { mainTex, fineTex };
}
PLAIN ENGLISH

注释：频谱颗粒生成技术

方法定义开始

步骤1：生成基础随机噪声

步骤2：高通滤波提取高频细节（细颗粒）

步骤3：如果需要软颗粒效果，进行模糊处理

步骤4：根据图像亮度混合颗粒（亮部少暗部多）

返回粗细两种颗粒纹理

🔬
为什么叫"频谱"颗粒？

真实胶片颗粒有不同的"尺寸"。高通滤波提取的是高频（细颗粒），模糊后的是低频（粗颗粒）。把不同频率的颗粒叠加，就能得到自然的颗粒结构——就像音频里的低音+高音组成完整音乐一样。
